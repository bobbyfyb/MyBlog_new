# MyBlog
## 概要：
  利用python3和django写的一个个人博客网站。比较简单，适合初学者练手。基于django自带的开发服务器和sqlite
  数据库，无需配置服务器和数据库。
  
## 运行环境：
  - python3.6
  - django 1.11.10
  
## 参考教程：
  - [Django博客教程](https://www.zmrenwu.com/post/2/ "django博客教程")
